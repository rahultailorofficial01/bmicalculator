PROJECT: BMI CALCULATOR WEB APPLICATION

DESCRIPTION:
This is a web-based Advanced BMI & Health Analyzer application. It calculates BMI, BMR, Ideal Weight, and Water Intake using standard health algorithms. It features a responsive UI and real-time health analysis.

HOW TO RUN THE PROJECT:
IMPORTANT: Ensure your computer is connected to the Internet.
(This is required to load the Design/CSS and Icons properly).
Open the project folder.
Double-click on the 'index.html' file.
The application will launch in your default web browser (Chrome, Edge, or Firefox).

FILES INCLUDED:
index.html  - Main application structure.
style.css   - Custom styling and animations.
script.js   - Calculation logic and formulas.
